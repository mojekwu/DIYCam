# DIYCam [![Build Status](https://travis-ci.org/mcndubuisi/diycam.svg?branch=master)](https://travis-ci.org/mcndubuisi/diycam)

## Installing

```
$ git clone https://github.com/mcndubuisi/diycam.git
$ cd diycam
$ npm install
$ npm start
```
